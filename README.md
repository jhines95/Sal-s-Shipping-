# Shipping Cost Calculator

A simple Python script that calculates shipping costs using three different shipping methods:

- Ground Shipping  
- Premium Ground Shipping  
- Drone Shipping

The script uses basic conditional logic to determine pricing based on package weight.

---

## Requirements

- Python 3.11+

---

## Usage

Run the script in a Python interpreter or save it as `shipping_calculator.py` and execute:

```bash
python shipping_calculator.py
